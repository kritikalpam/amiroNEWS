# **App Name**: Amironews Viewer

## Core Features:

- Website Display: Displays the amironews.com website in a full-screen WebView.
- Splash Screen: Provides a splash screen for 2-3 seconds with the logo and a background color before loading the website.
- WebView Navigation: Handles navigation within the WebView. Pressing back navigates back in WebView history, unless it's the homepage.
- Link Handling: Prevents links clicked within the WebView from opening in an external browser.
- Permissions: Request internet permissions on the Android platform.

## Style Guidelines:

- Primary color: Deep Blue (#2C417A), inspired by the logo, for a sense of authority and calm.
- Background color: Very light blue-gray (#E8EAF6), close in hue to the primary but desaturated and light to ensure readability.
- Accent color: Muted Red (#A93232), analogous to the primary but provides contrast for key actions.
- Body and headline font: 'PT Sans', a modern, readable sans-serif font for a clean and accessible reading experience.
- Minimal iconography; the focus is on the news content.
- Full-screen WebView to eliminate distractions and extra UI elements, ensuring an immersive reading experience.
- Simple fade-in animation for the splash screen to provide a smooth transition.
