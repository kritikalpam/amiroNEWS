# **App Name**: Amironews Reader

## Core Features:

- Content Display: Display the amironews.com website content in a simplified, reader-friendly format, optimized for mobile viewing.
- Website Navigation: Enable users to navigate the amironews.com website directly within the app.
- Studio Display: Display https://studio.firebase.google.com/studio-6936726665 Firebase Studio page content

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF), to create a calm and trustworthy atmosphere.
- Background color: Very light blue (#F0F8FF) which offers a subtle contrast.
- Accent color: Light purple (#BDB2FF), used for interactive elements such as buttons and links.
- Font pairing: 'Literata' (serif) for headlines and 'Inter' (sans-serif) for body text.
- Use minimalist icons for navigation and content categories.
- Employ a clean, single-column layout for easy reading and navigation.
- Subtle transitions when navigating between articles.
